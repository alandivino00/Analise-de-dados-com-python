# Análise de Cancelamento de Clientes com Python

## 📌 Sobre o Projeto
Este projeto tem como objetivo realizar uma **análise de dados sobre cancelamento de clientes ** utilizando **Python**. A partir de uma base de dados contendo informações sobre clientes de uma empresa fictícia, foram feitas análises para identificar **padrões de cancelamento e possíveis fatores que influenciam essa decisão**.

O projeto demonstra como a **análise de dados pode auxiliar empresas na tomada de decisões**, permitindo identificar problemas e propor estratégias para reduzir a taxa de cancelamento de clientes.

## 🎯 Objetivo
- Analisar o percentual de cancelamento de clientes.
- Identificar os principais fatores associados ao cancelamento.
- Explorar os dados para gerar insights que possam ajudar na **redução da taxa de churn**.

## 📊 Base de Dados
A base de dados utilizada contém informações de clientes de uma empresa fictícia. Entre os principais dados disponíveis estão:

- Idade
- Sexo
- Tempo como cliente
- Frequência de uso do serviço
- Número de ligações para o call center
- Dias de atraso
- Tipo de assinatura
- Duração do contrato
- Total gasto
- Meses desde a última interação
- Indicador de cancelamento (0 = não cancelou, 1 = cancelou)

## ⚙️ Tecnologias Utilizadas

- Python
- Pandas
- Plotly
- Jupyter Notebook
- VS Code

## 📈 Etapas do Projeto

1. Importação e exploração da base de dados
2. Tratamento e limpeza dos dados
3. Análise exploratória
4. Criação de gráficos para visualização
5. Identificação de padrões relacionados ao cancelamento
